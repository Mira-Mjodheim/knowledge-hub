# Contributing to KnowledgeHub

Merci de l'intérêt que vous portez à KnowledgeHub ! Pour contribuer, veuillez suivre ces étapes :

## Comment contribuer ?

1.  **Forkez** le dépôt.
2.  Créez une **branche** pour votre fonctionnalité (`git checkout -b feature/AmazingFeature`).
3.  **Commitez** vos changements (`git commit -m 'Add some AmazingFeature'`).
4.  **Poussez** vers la branche (`git push origin feature/AmazingFeature`).
5.  Ouvrez une **Pull Request**.

## Rapports de bugs

Si vous trouvez un bug, veuillez ouvrir une issue en décrivant le problème, les étapes pour le reproduire et votre environnement.

## Standards de code

Veuillez respecter les conventions de codage existantes (ESLint, Prettier si configurés) et ajouter des tests pour toute nouvelle fonctionnalité.
